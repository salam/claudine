## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| **Ctrl+Shift+K** | Open board |
| **Ctrl+Shift+N** | New conversation |
| **Ctrl+Shift+F** | Search conversations |
| **Ctrl+Shift+I** | Show needs-input |
| **Ctrl+Shift+R** | Refresh board |
| **Ctrl+Shift+C** | Focus Claude tab |

## More Features

- **Drag and drop** cards between columns to update status
- **Search** across conversation titles, descriptions, and JSONL content
- **Filter** by category (Bug, Feature, Task, etc.)
- **Export** the board to PDF, CSV, XML, or Trello/Jira format
