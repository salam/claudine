Add scrollable toolbar and panel title bar actions

- Sidebar toolbar: vertically scrollable when panel height is too small
  to show all buttons; thin 4px scrollbar with VS Code theme colors
- Panel title bar actions: new `claudine.toolbarLocation` setting
  (sidebar | titlebar | both) to show toolbar buttons in the VS Code
  panel header row, like Debug Console; webview sidebar hides in
  titlebar-only mode
- New commands: toggleSearch, toggleFilter, toggleCompactView,
  toggleExpandAll, toggleArchive — forwarded from title bar to webview
  via toolbarAction message type
- view/title menu contributions with codicon icons, gated by
  toolbarLocation setting via `when` clauses
- Sidebar brand: panel view shows only elephant icon; sidebar/sidedock
  shows only vertical "Claudine" text (avoids redundancy with VS Code
  native titles)
