Add walkthrough, webview auth, zoom controls, resizable columns

- Walkthrough: 5-step onboarding via contributes.walkthroughs with markdown
  media files; first-run notification offers walkthrough or direct open
- Webview origin validation: per-session auth token injected into HTML and
  attached to every postMessage; extension rejects messages with invalid token
- Zoom: sidebar buttons (-, %, +), keyboard shortcuts (Ctrl+=/-/0), CSS zoom
  property, range 50%–150% in 10% steps, persisted via webview state
- Resizable columns: drag handles between columns (horizontal layout only),
  getBoundingClientRect-based width calculation with zoom compensation,
  double-click to reset, persisted via webview state
- Shared mergeState() helper in vscode.ts for safe concurrent persistence
